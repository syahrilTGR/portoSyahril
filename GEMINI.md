# Web Portfolio Context - Muhammad Syahril Eka Pratama

## Project Overview
This repository contains the personal web portfolio for Muhammad Syahril Eka Pratama, highlighting his background in Telecommunications and Digital Technology (Politeknik Negeri Malang) and his notable projects like "HiJack" (IoT Smart Jacket).

## Development Roadmap
### Phase 1: Vanilla Foundation (Current)
- **Stack**: HTML5, Vanilla CSS3, Vanilla JavaScript.
- **Objective**: Build a highly performant, lightweight, and visually stunning 1-page portfolio. Focus heavily on layout, design system, and micro-animations.
- **Aesthetics**: Dark mode, glassmorphism, glowing gradients, modern typography (Outfit/Inter).

### Phase 2: Next.js Migration (Future)
- **Stack**: Next.js, React, TypeScript.
- **Objective**: Migrate the Vanilla project to a robust Next.js framework.
- **Why**: To align with enterprise-level standards (like Benedictus Pascal Kaligis's portfolio), improve SEO, and allow for scalable dynamic content (e.g., a blog or detailed project case studies) in the future. The CSS and logic from Phase 1 will be converted into reusable React components.

## Guidelines for AI Agent
1. **Always prioritize aesthetics**: Ensure the UI looks premium and doesn't feel "generic."
2. **Modular Code**: Even in Vanilla, write CSS and JS in a way that is easy to extract into React components later (e.g., clear class names, separated CSS variables).
3. **No Tailwind**: Use pure CSS for styling to maximize flexibility during Phase 1.
