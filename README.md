## 📌 Overview
This repository contains the source code for my personal web portfolio. It serves as a digital showcase of my engineering capabilities, spanning from hardware prototyping and IoT architectures to AI models and deep-level operating system configurations.

## 🚀 Featured Experience
The portfolio highlights end-to-end technical projects, including:
1. **HiJack (Smart Jacket Anti-Hipotermia):** IoT wearable funded by PKM 2025 with dual HAKI certifications.
2. **Kido Smart Ecosystem:** Integrated ESP32 hardware terminal and Kotlin Android app.
3. **Eco-Bin:** Distributed AI smart waste classification using MQTT and edge computing.
4. **Automotive Systems:** ESP32-based Quick Shifter & Launch Control.
5. **CaloriLens:** Computer vision model for food recognition and calorie tracking.
6. **Advanced Systems Configuration:** Engineering native macOS environments on unsupported PC hardware (Hackintosh) via deep ACPI/DSDT patching and Kernel extensions.

## 🛠️ Technology Stack (Phase 1)
Currently, this portfolio is built with a focus on performance, lightweight footprint, and a modern aesthetic.
- **HTML5** for semantic structure.
- **Vanilla CSS3** for styling, implementing a custom **Glassmorphism** design system with dynamic gradients and micro-animations.
- **Vanilla JavaScript** for interactive elements like scroll-reveal animations.

*(Note: No CSS frameworks like Tailwind were used in Phase 1 to maximize flexibility and custom aesthetic control).*

## 🗺️ Roadmap
**Phase 2:** Migration to **Next.js & TypeScript**.
This future update will componentize the current Vanilla UI into React components to align with enterprise-level scalable standards and enable dynamic content generation (like a technical blog).

## 💻 How to Run
Since the current version is a static build, simply clone this repository and open `index.html` in your favorite modern web browser. No `npm install` or build steps are required.

```bash
git clone https://github.com/syahrilTGR/portoSyahril.git
cd portoSyahril
open index.html
```
